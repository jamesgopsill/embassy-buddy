# Components

The buddy board features the following components.

- Ethernet
- EEPROM
- Filament Sensor
- 2 Fans
  - Hotend
  - Layer
- LCD
- Rotary Push Button & Encoder
- 4 Stepper Motors (Powered by TMC2209 Drivers)
  - X
  - Y
  - Z
  - E
- 4 Thermistors (Pull-Up Arrangement)
  - Bed
  - Board
  - Hotend
  - Pinda
- PINDA
- 2 x USB
  - USB serial
  - USB storage
